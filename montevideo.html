Rails Girls Montevideo 21-22 de Mayo del 2014

Hola Mundo!

Rails Girls llega a Montevideo, Uruguay por primera vez este 21-22 de Mayo: Durante este evento gratuito de 2 días exploraremos el interesante mundo de crear aplicaciones web con Ruby on Rails.

Inscripciones cerradas

Tweet

Aprenderás diseño, prototipeo y programación con ayuda de nuestros coaches.

Requieres una laptop, curiosidad y una pizca de imaginación! No necesitas saber programar pero sí muchas ganas de aprender :)
